== Exam ==
1. Create a custom post type, use create_custom_post_type.php and write your code inside the file given.
2. Add Taxonomy on the custom post type you created.
3. Add metafield box on the custom post type you created and display inside the template ( you can use any name for the field. make sure to display it inside cpt template ).
4. Add options to customize the Chart color and data.
    a. Option to change background color of chart. ( you can use wp color picker or any library ).
    b. Option to change the border color of chart.
    b. Options to change the data on the chart with your custom post type options.
